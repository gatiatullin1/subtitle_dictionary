# Rezka Subtitle Dictionary

Расширение для Chrome, которое перехватывает субтитры на hdrezka.ag,
переводит строки и слова, и копит личный словарь с карточками для повторения.

## Как устроено

- **content.js** — следит за блоком `#pjs_cdnplayer_subtitle` в плеере rezka
  (плеер рисует текущую строку субтитров обычным текстом в DOM, без `<track>`)
  и шлёт каждую новую строку в фон расширения.
- **background.js** — хранит последние 50 строк в памяти, пока вкладка открыта.
- **popup.html/js/css** — окно, которое открывается по клику на иконку:
  вкладки «Субтитры», «Словарь», «Повторение», «Статистика».
- Перевод идёт через публичный (без ключа) endpoint Google Translate.
  Это неофициальный, недокументированный путь — обычно стабильно работает,
  но Google может в теории ограничить его без предупреждения.

## Установка (режим разработчика)

1. Открой Chrome → адресная строка: `chrome://extensions`
2. Включи переключатель **«Режим разработчика»** (правый верхний угол)
3. Нажми **«Загрузить распакованное расширение»**
4. Укажи папку `rezka-extension` (ту, где лежит `manifest.json`)
5. Расширение появится в списке и в панели иконок браузера (может быть скрыто
   под иконкой «пазл» — закрепи его оттуда)

## Как пользоваться

1. Открой любой фильм/серию на **hdrezka.ag**, включи субтитры в плеере
2. Кликни на иконку расширения — откроется popup
3. Вкладка **«Субтитры»**: здесь появляются строки по мере воспроизведения
   - кнопка **«Перевести»** — переводит конкретную строку
   - 🔊 — озвучивает строку (английское произношение через системный TTS)
   - **клик по отдельному слову** — добавляет его в словарь вместе с переводом
     и контекстом (вся фраза, в которой слово встретилось)
4. Вкладка **«Словарь»**: список всех сохранённых слов, поиск, удаление,
   озвучка отдельного слова
5. Вкладка **«Повторение»**: карточки spaced repetition (упрощённый SM-2) —
   видишь слово → жмёшь, чтобы открыть перевод → оцениваешь
   «забыл / сложно / хорошо / легко» → карточка возвращается через
   правильный интервал
6. Вкладка **«Статистика»**: сколько слов в словаре, сколько повторений
   сделано, сколько ждут повторения сегодня

## Известные ограничения

- Popup закрывается, если кликнуть на саму страницу/видео — это стандартное
  поведение Chrome, расширения не могут это изменить. Субтитры продолжают
  копиться в фоне, даже когда popup закрыт — просто открой его снова, чтобы
  увидеть свежие строки.
- Если rezka изменит структуру плеера (id контейнера субтитров), скрипт
  перестанет находить строки — потребуется поправить селектор в `content.js`.
- Работает только на `hdrezka.ag` (и `rezka.ag`). Если у тебя другое зеркало
  (другой домен), добавь его в `matches` в `manifest.json` и в
  `host_permissions`.
- Перевод и словарь хранятся локально в браузере (`chrome.storage.local`),
  между компьютерами не синхронизируются.

## Если что-то не работает

Открой DevTools на странице rezka (F12) → вкладка Console — там должны быть
видны ошибки, если `content.js` не находит контейнер субтитров или перевод
не отвечает.
